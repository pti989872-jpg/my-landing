# RetroBoard Live

Первый живой вариант ретро-доски без внешних сервисов и npm-зависимостей.

## Что уже есть

- вход по имени и коду комнаты;
- режим инкогнито;
- общая доска Mad/Sad/Glad;
- добавление стикеров;
- перетаскивание стикеров между колонками;
- синхронизация между несколькими браузерами;
- реакции;
- назначение решения с ответственным и сроком;
- сохранение данных в `data/board.json`.

## Запуск

```powershell
cd "C:\Users\tpolyakova\Desktop\VS Code First Steps\retro-live"
.\start_server.ps1
```

После запуска откройте:

- на этой машине: `http://localhost:8080`
- с компьютеров коллег в той же сети: `http://IP_ВАШЕГО_КОМПЬЮТЕРА:8080`

Код комнаты по умолчанию: `retro2026`.

Если коллеги не могут зайти, откройте входящий порт в Windows Firewall:

```powershell
.\open_firewall_admin.ps1
```

Скрипт нужно запускать из PowerShell с правами администратора.
Можно также запустить файл `open_firewall_admin.cmd` правой кнопкой мыши: **Run as administrator**.

Остановить сервер:

```powershell
.\stop_server.ps1
```

## Важно

Это не промышленная авторизация, а первый рабочий слой для пилота. Для реального внедрения лучше заменить код комнаты на корпоративную авторизацию или хотя бы роли ведущий/участник.
